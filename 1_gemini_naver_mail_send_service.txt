url : https://gemini-chat-assistant.replit.app/
사용 도구 : Replit

바이드 코딩 도구인 Replit 을 사용하여 gemini api 로 답변 받은 내용을 자동으로 naver 메일로 전송하는 웹 사이트를 구현함.
코드는 단 한글자도 작성하지 않았고, 오로지 프롬프트로만 해당 웹페이지 개발 및 테스트, 배포까지 완료.

후기 : 
1. 바이브 코드 말의 유래 처럼, 코드라는 것이 존재하는지도 모르고 오로지 자연어 명령과 실제 동작에만 집중할 수 있었다.
2. 프론트의 경우에는 정말 개발 지식이 없는 사람도 할 수 있을 정도로 좋은 퀄리티로 뚝딱 개발됨.
3. 단 백앤드에서의 복잡한 로직이나, 다수의 트래픽 및 사용자 처리를 프롬프팅만으로 처리하기에는 한계가 있어보인다.
4. 배포도 정말 버튼 하나 딸깍으로 순식간에 배포가 완료되었다...
5. 위 페이지 개발, 테스트, 배포 전체 과정이 3시간만에 끝났음.
